## 图

### 图的建立

邻接矩阵
* 使用一个n*n的矩阵来描述图
* 二维数组表示
* 无权图，n[i][j] 表示i到j之间有无边，从i指向j。
* 对于有权图，n[i][j]中可以存储权重

邻接表
* 对于每个点，存储一条链表，用来指向所有与该点直接相连的点
* 对于有权图，链表中值对应权重

### 图的遍历

* 深度优先遍历。从图中某顶点出发，不断访问邻居，邻居的邻居，直到访问完毕
* 广度优先遍历
* 可参考 841 题
