## 栈
后进先出

常见操作
* 进栈 - push -> O(1)
* 出栈 - pop -> O(1)
* 取栈顶 - top -> O(1)
* 判断是否为空栈 - isEmpty -> O(1)

应用
* 深度优先遍历的基础
* 函数调用栈
* 浏览器前进后退
* 匹配括号
* 单调栈寻找下一个更大（小）元素

小知识 -> 波兰表示法(前缀)和逆波兰表示法(后缀)
* 逻辑、算术和代数表示方法
* 操作符的位置（+ 3 4）-> （3 4 +）-> (3 + 4)
* 逆波兰表达式可用栈计算（遇到数字就将数字压栈，遇到操作符，就将栈顶的两个元素取出计算，将计算结果再压入栈）(题目150)
